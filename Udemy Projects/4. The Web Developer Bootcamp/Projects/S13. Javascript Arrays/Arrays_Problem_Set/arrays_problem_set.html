1. printReverse()

Write a function that takes an array as an argument and prints out the elements in the array in reverse order(dont actually reverse the array itself)

  • printReverse([1,2,3,4]); = 4,3,2,1
  • printReverse(["a","b","c"]); = "c","b","a"

2. isUniform()

Write a function which takes an array as an argument and returns true if all elements in the array are identical

  • isUniform([1,1,1,1]); = true
  • isUniform([2,1,1,1]); = false
  • isUniform(["a","b","p"]); = false
  • isUniform(["b","b","b"]); = true

3. sumArray()

Write a function that accepts an array of numbers and returns the sum of all numbers in the array

  • sumArray([1,2,3]); = 6
  • sumArray([10,3,10,4]); = 27
  • sumArray([-5,100]); = 95

4. max()

Write a function that accepts an array of numbers and returns the maximum number in the array

  • max([1,2,3]); = 3
  • max([10,3,10,4]); = 10
  • max([-5,100]); = 100
